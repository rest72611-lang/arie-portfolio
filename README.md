# Arie Steinberg Portfolio UI

A static one-page portfolio website for Arie Steinberg.

## Includes

- Home / About / Featured Projects / Skills / Focus / Contact
- Five polished portfolio projects
- Real GitHub / npm / live demo links
- John Bryce certificate link
- n8n-ready contact form placeholder

## How to use

Open `index.html` in the browser.

For deployment, upload the folder to GitHub Pages, Netlify, Vercel, Cloudflare Pages, or any static hosting.

## Contact form

Open `script.js` and replace:

```js
const N8N_WEBHOOK_URL = "PASTE_YOUR_N8N_WEBHOOK_URL_HERE";
```

with your real n8n webhook URL.
